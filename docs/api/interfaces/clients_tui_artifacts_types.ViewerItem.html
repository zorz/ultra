    Interface ViewerItem

    Generic item interface for any viewer. This is the minimal interface that all viewer items must implement. More specialized viewers can extend this with additional properties.

    Example

    // Simple list viewer item
    interface FileItem extends ViewerItem {
      path: string;
      size: number;
    }

    // Tree viewer item (uses children)
    interface OutlineItem extends ViewerItem {
      symbolKind: string;
      range: Range;
    }
