    • create(
          command: string,
          args: string[],
          workspaceRoot: string,
      ): Promise<LSPClient | null>

      Create and initialize an LSP client

      Factory method that creates the client and starts the language server. Use this instead of constructor + start() for cleaner async initialization.

      Parameters

      • command: string

        The language server command

      • args: string[]

        Arguments to pass to the language server

      • workspaceRoot: string

        The workspace root path

      Returns Promise<LSPClient | null>

      Initialized LSP client, or null if initialization failed

      Example

      const client = await LSPClient.create(
        'typescript-language-server',
        ['--stdio'],
        '/path/to/workspace'
      );
      if (client) {
        // Client is ready to use
      }
