    Class InputDialog

    Base class for Promise-based dialogs.

    Extends BaseDialog with:

    • Promise-based lifecycle (resolves on close)
    • Close reason tracking (confirm/cancel/dismiss)
    • Result value passing
    • Auto-centering on screen
