# Welcome to Ultra

A terminal-native code editor built with TypeScript and Bun.

## Quick Start

- **Ctrl+P** - Quick Open (find files)
- **Ctrl+Shift+P** - Command Palette
- **Ctrl+`** - Toggle Terminal
- **Ctrl+B** - Toggle Sidebar

## Getting Help

- Press **Ctrl+Shift+P** and type "help" to see available commands
- Visit https://github.com/zorz/ultra for documentation
